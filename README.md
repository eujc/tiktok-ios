### 如何更改国家和地区：
默认是日本区，可以更改这行代码中的JP (?<=_region=)CN(?=&) JP 307<br>
美国示例： (?<=_region=)CN(?=&) US 307<br>
英文简写 JP（日本）｜KR（韩国）｜UK（英国）｜US（美国）｜TW（中国台湾）<br>
